# TriviaGame

<h1>Big Bang Theory Trivia Game</h1>

<ul>
<li>This is a game to test your knowledge of the hit sitcom The Big Bang Theory.</li>
<li>You will be asked a question and you then have 15 seconds to chose the right. answer for the 4 options.</li>
<li>After all the questions are done, you will be given a summary of how well you did.</li>
<li>The game is made using HTML5, CSS3, Javascript, and JQuery.</li>
</ul>
